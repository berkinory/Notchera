# notchera 0.1.0

## features

- restored the match album art color setting in media settings.

## improvements

- improved the onboarding flow. the onboarding window now stays in front during setup, and the notch can no longer expand until onboarding is complete.
- refined the menubar icon with a larger rounded shape and a smoother white-gray gradient.

## bug fixes

- fixed the expanded media empty state. when nothing is playing, the home view now shows a proper not playing placeholder instead of broken, empty, or unknown metadata.
